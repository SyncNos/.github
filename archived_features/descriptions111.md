### 📌 Short description (Summary, 132 char limit)


### 📄 Detailed description (Description)

```
SyncNos is an open-source web clipper that saves whatever you find useful in the browser — straight into Notion or Obsidian.

━━━━━━━━━━━━━━
📥 What it captures
━━━━━━━━━━━━━━
• AI chats: ChatGPT, Gemini, Claude, and 11+ major AI platforms worldwide
• Web articles: one-click extraction from any page, with dedicated support for WeChat Official Accounts and Xiaohongshu image posts
• Video transcripts: YouTube and Bilibili, with full timestamps

━━━━━━━━━━━━━━
📤 Where it syncs
━━━━━━━━━━━━━━
• Notion: one-click OAuth, auto-creates the database
• Obsidian: via the Local REST API plugin, fully local
• Local export: Markdown / Zip backup

━━━━━━━━━━━━━━
✨ Handy extras
━━━━━━━━━━━━━━
• Inline comments — highlight, annotate, and sync alongside your content
• Type "$" in any AI chat to invoke saved items
• Insight panel for a full overview of your local data

━━━━━━━━━━━━━━
🔒 Privacy
━━━━━━━━━━━━━━
Data lands in your own browser first — you decide where it syncs. Fully open source, exportable anytime.

GitHub: https://github.com/chiimagnus/SyncNos
```
